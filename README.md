📌 Overview

WebCheck360 is an advanced Website Intelligence & Monitoring Platform developed to analyze websites for:

⚡ Performance Issues
🔗 Broken Links
🛡️ Security Risks
🎭 Phishing Indicators
🌐 Trust & Authenticity
🤖 AI-Based Recommendations

The system combines traditional website analysis techniques with AI-powered interpretation to provide a complete Website Risk Intelligence Dashboard.

✨ Features
🔗 Website Link Scanner
Extracts all links from a website
Detects broken links
Identifies suspicious links
Calculates website health statistics
⚡ Website Speed Analyzer

Measures:

Page Load Time
Server Response Time (TTFB)
Page Size
Performance Score
🛡️ Risk Intelligence Engine

Generates risk scores using:

Broken link ratio
Performance score
HTTPS usage
Server response time
Page structure analysis
Risk Levels
Score Range	Risk Level
0 – 30	✅ SAFE
31 – 60	⚠️ MODERATE
61 – 100	🚨 HIGH RISK
🎭 Trust & Phishing Detection Engine

Detects phishing websites using:

HTTPS validation
Domain age analysis
URL entropy detection
Login form detection
Hidden iframe detection
JavaScript redirect detection
External form action analysis
🤖 AI Insight Engine

Integrated with OpenRouter GPT-3.5

Provides:

AI-generated analysis

Performance optimization tips
Human-readable risk explanations
📊 Monitoring Dashboard

Displays:

Risk trends

Interactive charts
💬 AI Chat Assistant

Users can:

⚙️ Installation & Setup
1️⃣ Clone Repository
git clone https://github.com/your-username/WebCheck360.git
2️⃣ Move Project to XAMPP

Place the project folder inside:

htdocs/
3️⃣ Start Services

Open XAMPP Control Panel and start:

Apache
MySQL
4️⃣ Import Database
Open phpMyAdmin
Create database:
webcheck360
Import the provided .sql file
5️⃣ Configure Database

Update database credentials in:

config/db.php

Example:

$host = "localhost";
$user = "root";
$password = "";
$database = "webcheck360";
6️⃣ Run Project

Open browser:

http://localhost/WebCheck360
